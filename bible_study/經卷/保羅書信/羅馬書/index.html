羅馬書研讀 Romans Study

福音的大能：從定罪到稱義，從死亡到生命

寫作背景

羅馬書是保羅在第三次宣教旅程中（約主後57年），於哥林多寫成的書信。當時保羅尚未到過羅馬教會，但計劃前往西班牙宣教，途經羅馬。這封書信是他最系統化、最完整的神學論述。

保羅第三次宣教旅程

保羅從以弗所出發，經過馬其頓、希臘，最後在哥林多停留三個月，期間寫成羅馬書。

寫作時間

約主後57年，保羅在哥林多停留三個月期間

寫作地點

哥林多（參羅16:23，該猶是哥林多教會的人）

寫作目的

向羅馬教會介紹自己，闡明福音的核心真理，為未來的宣教旅程鋪路

歷史背景

羅馬教會由猶太和外邦信徒組成，需要處理律法與恩典、猶太人與外邦人的關係

分段結構

一、引言與主題（1:1-17）

保羅的自我介紹、問候，以及全書的主題宣告：「我不以福音為恥；這福音本是神的大能，要救一切相信的，先是猶太人，後是希臘人。因為神的義正在這福音上顯明出來；這義是本於信，以至於信。」

二、全人類的定罪（1:18-3:20）

論證全人類都在罪中，都需要神的救恩：外邦人的罪（1:18-32）、猶太人的罪（2:1-3:8）、全人類的結論（3:9-20）。

三、因信稱義的福音（3:21-5:21）

闡明因信稱義的真理：稱義的方法（3:21-31）、稱義的例證（4:1-25）、稱義的果效（5:1-21）。

四、成聖的生活（6:1-8:39）

論述稱義後的生活：與罪斷絕（6:1-23）、與律法的關係（7:1-25）、在聖靈裡的生活（8:1-39）。

五、神對以色列的計劃（9:1-11:36）

解釋以色列的過去（9:1-29）、現在（9:30-10:21）、將來（11:1-36）。

六、實際生活的應用（12:1-15:13）

將福音真理應用於生活：獻身的生活（12:1-2）、教會生活（12:3-13:14）、軟弱與剛強（14:1-15:13）。

七、結語（15:14-16:27）

保羅的計劃、問候與頌讚。

核心議題討論

1. 同性戀（1:26-27）

經文：

「因此，神任憑他們放縱可羞恥的情慾。他們的女人把順性的用處變為逆性的用處；男人也是如此，棄了女人順性的用處，慾火攻心，彼此貪戀，男和男行可羞恥的事，就在自己身上受這妄為當得的報應。」（羅1:26-27）

觀察（Observation）

  • • 這段經文出現在論述「外邦人的罪」的脈絡中
  • • 保羅描述的是「放縱可羞恥的情慾」的行為
  • • 用「順性」與「逆性」來對比正常與異常的關係
  • • 指出這是「妄為當得的報應」
  • • 上下文顯示這是拒絕認識神的後果之一

解釋（Interpretation）

  • • 保羅明確指出同性性行為違反神的創造秩序
  • • 「順性」指神在創造中設立的男女關係
  • • 這不是針對性傾向，而是針對性行為
  • • 這是人拒絕神、放縱私慾的表現之一
  • • 需要以恩典和真理並重的方式回應

應用（Application）

  • • 持守聖經對性行為的標準，同時以愛心對待所有人
  • • 認識到所有人都是罪人，都需要神的恩典
  • • 在教會中提供支持，幫助有掙扎的人
  • • 避免歧視，但也不妥協真理
  • • 強調悔改和聖靈更新的可能性

2. 因信稱義（3:21-5:21）

核心經文：

「但如今，神的義在律法以外已經顯明出來，有律法和先知為證：就是神的義，因信耶穌基督加給一切相信的人，並沒有分別。因為世人都犯了罪，虧缺了神的榮耀；如今卻蒙神的恩典，因基督耶穌的救贖，就白白的稱義。」（羅3:21-24）

觀察（Observation）

  • • 「但如今」標誌著從定罪到稱義的轉折
  • • 稱義是「在律法以外」，不靠行為
  • • 稱義的基礎是「因信耶穌基督」
  • • 稱義的範圍是「一切相信的人」
  • • 稱義是「白白的」，是恩典

解釋（Interpretation）

  • • 稱義是法律用語，指被宣告為義
  • • 不是因為我們的行為，而是因為基督的義歸算給我們
  • • 信心是接受這恩典的管道，不是功勞
  • • 亞伯拉罕是信心的典範（4:1-25）
  • • 稱義帶來與神和好、進入恩典中（5:1-2）

應用（Application）

  • • 放下靠行為得救的嘗試，單單信靠基督
  • • 在得救的確據中得享平安和喜樂
  • • 避免律法主義和反律主義兩個極端
  • • 以感恩的心回應神的恩典
  • • 在教會中強調福音的核心地位

3. 浸禮（6:1-11）

核心經文：

「所以，我們藉著洗禮歸入死，和他一同埋葬，原是叫我們一舉一動有新生的樣式，像基督藉著父的榮耀從死裡復活一樣。我們若在他死的形狀上與他聯合，也要在他復活的形狀上與他聯合。」（羅6:4-5）

觀察（Observation）

  • • 保羅用「歸入死」和「一同埋葬」描述洗禮
  • • 洗禮與基督的死和復活緊密相連
  • • 洗禮指向「新生的樣式」
  • • 強調「在死的形狀上與他聯合」
  • • 洗禮是與基督聯合的記號

解釋（Interpretation）

  • • 浸禮（全身入水）最能表達「埋葬」的意義
  • • 洗禮是外在記號，表明內在的屬靈事實
  • • 不是洗禮本身拯救人，而是所代表的與基督聯合
  • • 洗禮是公開見證，宣告與舊生命斷絕
  • • 洗禮指向未來復活的盼望

應用（Application）

  • • 強調浸禮的重要性，作為順服的見證
  • • 幫助信徒理解洗禮的屬靈意義
  • • 提醒已受洗的信徒活出「新生的樣式」
  • • 洗禮前要有充分的教導和準備
  • • 洗禮是教會的重要見證時刻

4. 未聽福音的人得救原則（1:18-2:16）

核心經文：

「原來，神的忿怒從天上顯明在一切不虔不義的人身上，就是那些行不義阻擋真理的人。神的事情，人所能知道的，原顯明在人心裡，因為神已經給他們顯明。自從造天地以來，神的永能和神性是明明可知的，雖是眼不能見，但藉著所造之物就可以曉得，叫人無可推諉。」（羅1:18-20）

「沒有律法的外邦人若順著本性行律法上的事，他們雖然沒有律法，自己就是自己的律法。這是顯出律法的功用刻在他們心裡，他們是非之心同作見證，並且他們的思念互相較量，或以為是，或以為非。」（羅2:14-15）

觀察（Observation）

  • • 神透過「所造之物」向所有人啟示自己
  • • 人「無可推諉」，因為有足夠的啟示
  • • 外邦人有「是非之心」和「律法的功用刻在心裡」
  • • 但保羅也指出所有人都「虧缺了神的榮耀」（3:23）
  • • 需要「因信耶穌基督」才能稱義（3:22）

解釋（Interpretation）

  • • 神透過普遍啟示（自然）向所有人顯明
  • • 但普遍啟示不足以帶來救恩
  • • 需要特殊啟示（福音）才能得救
  • • 神的主權和人的責任需要平衡理解
  • • 最終的判斷在於神，我們要忠於傳福音的使命

應用（Application）

  • • 不要因為神的主權而忽略傳福音的責任
  • • 相信神是公義的，會按各人的情況審判
  • • 專注於完成大使命，傳揚福音
  • • 為未得之民禱告，支持宣教事工
  • • 在神的主權和人的責任之間保持平衡

5. 信主之後仍會犯罪的出路（7:14-8:2）

核心經文：

「我也知道在我裡頭，就是我肉體之中，沒有良善。因為，立志為善由得我，只是行出來由不得我。故此，我所願意的善，我反不做；我所不願意的惡，我倒去做。若我去做所不願意做的，就不是我做的，乃是住在我裡頭的罪做的。我覺得有個律，就是我願意為善的時候，便有惡與我同在。因為按著我裡面的意思，我是喜歡神的律；但我覺得肢體中另有個律和我心中的律交戰，把我擄去，叫我附從那肢體中犯罪的律。我真是苦啊！誰能救我脫離這取死的身體呢？感謝神，靠著我們的主耶穌基督就能脫離了。」（羅7:18-25）

「如今，那些在基督耶穌裡的就不定罪了。因為賜生命聖靈的律，在基督耶穌裡釋放了我，使我脫離罪和死的律了。」（羅8:1-2）

觀察（Observation）

  • • 保羅描述信主後仍有的掙扎
  • • 「立志為善由得我，只是行出來由不得我」
  • • 承認「我肉體之中，沒有良善」
  • • 有「兩個律」在交戰
  • • 出路是「靠著我們的主耶穌基督」

解釋（Interpretation）

  • • 信主後仍有罪性，但不再被罪轄制
  • • 聖靈住在我們裡面，提供得勝的能力
  • • 「在基督耶穌裡就不定罪了」— 地位已改變
  • • 成聖是漸進的過程，需要持續依靠聖靈
  • • 第八章強調聖靈的工作和得勝

應用（Application）

  • • 承認自己的軟弱，不因犯罪而絕望
  • • 依靠聖靈的能力，不靠自己的努力
  • • 記住在基督裡「不定罪」的地位
  • • 持續認罪、悔改，追求成聖
  • • 在教會中彼此扶持，共同成長

6. 外邦人數滿足猶大家全家得救（11:11-32）

核心經文：

「我且說，他們失腳是要他們跌倒嗎？斷乎不是！反倒因他們的過失，救恩便臨到外邦人，要激動他們發憤。若他們的過失為天下的富足，他們的缺乏為外邦人的富足，何況他們的豐滿呢？」（羅11:11-12）

「弟兄們，我不願意你們不知道這奧秘，恐怕你們自以為聰明，就是以色列人有幾分是硬心的，等到外邦人的數目添滿了，於是以色列全家都要得救。如經上所記：必有一位救主從錫安出來，要消除雅各家的一切罪惡。」（羅11:25-26）

觀察（Observation）

  • • 以色列的「失腳」帶來外邦人的機會
  • • 「等到外邦人的數目添滿了」
  • • 「以色列全家都要得救」
  • • 這是「奧秘」，不是人的推測
  • • 神的計劃包括猶太人和外邦人

解釋（Interpretation）

  • • 神有特定的時間表，外邦人得救有「數目」
  • • 「以色列全家」可能指餘民或全體
  • • 這不意味著所有猶太人都自動得救
  • • 需要透過相信耶穌基督
  • • 神的計劃最終要拯救祂的選民

應用（Application）

  • • 為猶太人歸主禱告，支持猶宣事工
  • • 不要輕看猶太人，他們是神的選民
  • • 積極向所有人傳福音，包括猶太人
  • • 相信神會按祂的時間完成計劃
  • • 在教會中接納不同背景的信徒

7. 順服在上位（13:1-7）

核心經文：

「在上有權柄的，人人當順服他，因為沒有權柄不是出於神的。凡掌權的都是神所命的。所以，抗拒掌權的就是抗拒神的命；抗拒的必自取刑罰。作官的原不是叫行善的懼怕，乃是叫作惡的懼怕。你願意不懼怕掌權的嗎？你只要行善，就可得他的稱讚；因為他是神的用人，是與你有益的。你若作惡，卻當懼怕；因為他不是空空地佩劍；他是神的用人，是伸冤的，刑罰那作惡的。所以你們必須順服，不但是因為刑罰，也是因為良心。你們納糧，也為這個緣故；因他們是神的差役，常常特管這事。凡人所當得的，就給他。當得糧的，給他納糧；當得稅的，給他上稅；當懼怕的，懼怕他；當恭敬的，恭敬他。」（羅13:1-7）

觀察（Observation）

  • • 「沒有權柄不是出於神的」
  • • 「抗拒掌權的就是抗拒神的命」
  • • 政府是「神的用人」
  • • 順服是「因為良心」，不只是因為刑罰
  • • 要納糧、上稅、恭敬掌權者

解釋（Interpretation）

  • • 神設立政府來維持秩序和公義
  • • 順服是原則，但有例外（違背神命令時）
  • • 需要平衡順服政府和順服神
  • • 當政府要求違背神時，要「順從神，不順從人」（徒5:29）
  • • 順服不等於認同所有政策

應用（Application）

  • • 在一般情況下順服政府和法律
  • • 履行公民責任（納稅、投票等）
  • • 為在上位的禱告（提前2:1-2）
  • • 當政府違背神時，以和平方式表達
  • • 在順服神和順服人之間保持智慧平衡

互動思考

思考問題 1

「因信稱義」如何影響你對自己得救確據的理解？

答案：因信稱義意味著我們的得救完全基於基督的工作，而不是我們的行為。這給我們確據，因為我們的救恩不依賴於自己的表現，而是依賴於神不變的應許。當我們軟弱時，可以確信神不會因為我們的行為而收回救恩。

思考問題 2

羅馬書7章的掙扎與8章的得勝，如何幫助你理解成聖的過程？

答案：第7章顯示即使信主後，我們仍會與罪掙扎，這是正常的。但第8章告訴我們，聖靈住在我們裡面，提供得勝的能力。成聖不是一次性的，而是持續依靠聖靈的過程。我們要記住在基督裡「不定罪」的地位，同時積極追求聖潔。

思考問題 3

如何在持守聖經標準的同時，以愛心對待有不同觀點的人？

答案：我們要「用愛心說誠實話」（弗4:15）。持守真理不等於缺乏愛心，表達愛心也不等於妥協真理。我們要以恩典和真理並重的方式，承認自己也是罪人，需要神的恩典，同時不否認聖經的標準。重點是關係和對話，而不是定罪和排斥。

思考問題 4

羅馬書12:1-2的「將身體獻上當作活祭」對你的日常生活有什麼實際意義？

答案：這意味著我們整個生命都要成為對神的敬拜。不是只在教會中，而是在工作、家庭、人際關係等各個領域都要活出神的旨意。要「心意更新而變化」，不被世界同化，而是以神的話語來塑造我們的思想和價值觀，好能「察驗何為神的善良、純全、可喜悅的旨意」。

"我不以福音為恥；這福音本是神的大能，要救一切相信的，先是猶太人，後是希臘人。" 羅馬書 1:16
