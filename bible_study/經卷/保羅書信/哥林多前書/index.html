哥林多前書研讀 1 Corinthians Study

教會生活的挑戰與智慧：在混亂中重建秩序，在紛爭中見證合一

寫作背景

歷史背景

哥林多前書是保羅在第三次宣教旅程中（約主後54-55年），於以弗所寫成的書信。當時保羅在以弗所事奉約三年（徒19:1-20:1），收到了從哥林多教會傳來關於教會內部問題的消息。

哥林多城位於希臘南部，是亞該亞省的省會，也是羅馬帝國最重要的商業港口之一。這座城市以其繁華、多元文化和道德敗壞而聞名。古代作家稱哥林多為「希臘的罪惡之都」，城中充滿了各種異教崇拜、商業活動和放縱的生活。

哥林多古城遺址示意圖

寫作原因

1. 回應教會的問題：保羅從革來氏家裡的人得知哥林多教會中有紛爭結黨的問題（1:11）。

2. 回答教會的詢問：哥林多教會曾寫信給保羅，詢問關於婚姻、吃祭偶像之物、屬靈恩賜等問題（7:1; 8:1; 12:1）。

3. 糾正錯誤的教導：保羅需要處理關於復活、主餐、屬靈恩賜等方面的誤解和濫用。

保羅與哥林多教會的關係

第一次到訪（主後50-52年）

保羅第二次宣教旅程中，在哥林多居住一年半，建立教會（徒18:1-18）

寫作前書（主後54-55年）

保羅在以弗所寫成哥林多前書，回應教會的問題和詢問

寫作後書（主後55-56年）

保羅寫成哥林多後書，進一步處理教會的關係和事工

紛爭結黨

教會內部有分門別類的現象

道德問題

亂倫、訴訟、道德放縱

教義混淆

對復活、主餐的誤解

分段結構

一、引言與問候（1:1-9）

保羅以自己使徒的身分和權柄，問候在哥林多的教會。他為神在他們身上的恩典和恩賜感謝神，並確信他們必能等候主再來。

關鍵經文：

「我常為你們感謝我的神，因神在基督耶穌裡所賜給你們的恩惠；又因你們在他裡面凡事富足，口才、知識都全備。」（1:4-5）

重點：

  • 保羅的使徒權柄來自神的呼召
  • 強調聖徒的身分和與基督的關係
  • 為神的恩典感謝，而非抱怨問題

二、處理紛爭結黨（1:10-4:21）

保羅嚴厲指責教會中的分門別類，強調只有基督是教會的根基，所有事工都是為神建造。

2.1 分門別類的問題（1:10-17）

教會中有人說「我是屬保羅的」、「我是屬亞波羅的」，保羅指出這種分黨是屬肉體的表現。

2.2 十字架的智慧（1:18-2:5）

神的愚拙總比人智慧，神的軟弱總比人強壯。十字架的信息看似愚拙，卻是神的大能。

2.3 真智慧與屬靈人（2:6-3:4）

屬靈的事要用屬靈的話解釋，屬靈的人能看透萬事，屬肉體的仍是嬰孩。

2.4 工人與根基（3:5-4:21）

保羅和亞波羅不過是執事，各人要照自己的工夫得賞賜。只有基督是根基。

三、處理道德問題（5:1-6:20）

3.1 亂倫事件（5:1-13）

有人收了繼母為妻，保羅要求教會執行紀律，將這人從他們中間趕出去，因為一點麵酵能使全團發起來。

3.2 信徒之間的訴訟（6:1-11）

信徒不應該彼此告狀，更不應該告到不信的人面前。聖徒要審判世界，豈能讓不義的人審判嗎？

3.3 身體的聖潔（6:12-20）

身體是聖靈的殿，不可與娼妓聯合。你們是重價買來的，要在你們的身子上榮耀神。

四、回答教會的詢問（7:1-15:58）

4.1 關於婚姻（7:1-40）

關於結婚、離婚、守獨身、寡婦等問題的教導。保羅強調各人要守住神的召，但要避免淫亂的事。

4.2 關於祭偶像之物（8:1-11:1）

討論知識與愛心的平衡。雖然偶像算不得什麼，但為軟弱弟兄的緣故，要限制自己的自由。

4.3 關於聚會和蒙頭（11:2-34）

論到聚會的秩序，包括婦女蒙頭的意義和主餐的正確態度。

4.4 關於屬靈恩賜（12:1-14:40）

聖靈的恩賜是為建立教會，愛是最大的恩賜。要追求愛，也要切慕屬靈的恩賜。

4.5 關於復活（15:1-58）

基督的復活是福音的核心，也是我們復活的保證。這是全書最長和最系統的教義論述。

五、結語（16:1-24）

關於為聖徒捐款、保羅的行程計劃、各種問候，以及最後的祝福和警告。

結束語：

「若有人不愛主，這人可詛可咒。主必要來！願主耶穌的恩常與你們眾人同在。」（16:22-23）

書信重點

1

教會的合一

保羅強調，教會不應該分門別類，因為「基督是分開的嗎？」（1:13）。所有事工都是為建造教會，最終的根基只有基督。

「因為我們是與神同工的；你們是神所耕種的田地，所建造的房屋。」（3:9）

  • 反對以領袖為中心的黨派
  • 強調基督是唯一的根基
  • 呼籲眾人合一，不可分黨
2

十字架的智慧

保羅顛覆了世界的智慧觀。十字架的信息看似愚拙，卻是神的大能；看似軟弱，卻勝過一切人的智慧。

「因為十字架的道理，在那滅亡的人為愚拙；在我們得救的人，卻為神的大能。」（1:18）

  • 神的愚拙總比人智慧
  • 神的軟弱總比人強壯
  • 十字架是神的智慧和能力
3

教會紀律

保羅要求教會執行紀律，將犯罪的人趕出去，保持教會的聖潔。「一點麵酵能使全團發起來」（5:6）。

「你們應當把那惡人從你們中間趕出去。」（5:13）

  • 維護教會的聖潔
  • 防止罪惡蔓延
  • 最終目的是挽回
4

愛是最妙的道

在討論屬靈恩賜的上下文中，保羅插入了一篇關於愛的偉大篇章。愛比所有的恩賜都重要，是最大的恩賜。

「如今常存的有信，有望，有愛這三樣，其中最大的是愛。」（13:13）

  • 愛是最大的恩賜
  • 愛是永不止息的
  • 愛是恩賜使用的動機
5

復活的真理

保羅詳細論證基督的復活和信徒的復活。如果沒有復活，信也是徒然，傳也是徒然（15:14）。

「若基督沒有復活，我們所傳的便是枉然，你們所信的也是枉然。」（15:14）

  • 復活是福音的核心
  • 基督的復活是初熟的果子
  • 信徒的復活是確定的
6

自由與責任

保羅討論基督徒的自由，但強調要為軟弱弟兄的緣故限制自己的自由。知識叫人自高自大，唯有愛心能造就人。

「只是你們要謹慎，恐怕你們這自由竟成了那軟弱人的絆腳石。」（8:9）

  • 自由要在愛中運用
  • 顧念軟弱的弟兄
  • 榮耀神是最高原則

原文難字解釋

以下是哥林多前書中一些重要的希臘原文詞彙及其意義：

σχίσμα (schisma)

中文譯詞：分裂、分門別類

出現位置：1:10; 11:18; 12:25

字義解釋：原指衣服的裂縫或撕裂，引申為團體中的分裂和分歧。保羅用此詞強烈指責哥林多教會的分黨現象。

經文應用：

「弟兄們，我藉我們主耶穌基督的名勸你們都說一樣的話。你們中間也不可分門別類，只要一心一意，彼此相合。」（1:10）

σοφία (sophia)

中文譯詞：智慧

出現位置：全書多次出現（1:17, 19, 20, 21, 22, 24, 30等）

字義解釋：保羅區分了兩種智慧：世界的智慧（屬人的）和神的智慧（屬天的）。十字架的道理是神的智慧，卻被世界看為愚拙。

經文應用：

「因為神的愚拙總比人智慧，神的軟弱總比人強壯。」（1:25）

πνευματικός (pneumatikos)

中文譯詞：屬靈的

出現位置：2:13, 15; 3:1; 9:11; 12:1; 14:1, 37; 15:44, 46

字義解釋：從「靈」（pneuma）而來，指與聖靈相關的事物或人。保羅區分屬靈的人與屬肉體的人。

經文應用：

「屬靈的人能看透萬事，卻沒有一人能看透了他。」（2:15）

σάρκινος / σαρκικός (sarkinos / sarkikos)

中文譯詞：屬肉體的

出現位置：3:1, 3, 4; 9:11

字義解釋：從「肉體」（sarx）而來，指被肉體和罪性控制的人，與屬靈的人相對。保羅指出哥林多信徒仍是屬肉體的嬰孩。

經文應用：

「弟兄們，我從前對你們說話，不能把你們當作屬靈的，只得把你們當作屬肉體，在基督裡為嬰孩的。」（3:1）

ἀγάπη (agape)

中文譯詞：愛

出現位置：第13章及全書多處

字義解釋：新約中最重要的愛字，指無條件的、犧牲的、神聖的愛。保羅在13章詳述這種愛的特質，並稱之為「最妙的道」。

經文應用：

「愛是恆久忍耐，又有恩慈；愛是不嫉妒；愛是不自誇，不張狂。」（13:4）

χαρίσματα (charismata)

中文譯詞：屬靈恩賜

出現位置：1:7; 7:7; 12:4, 9, 28, 30, 31

字義解釋：從「恩典」（charis）而來，指神所賜的各種能力和恩賜，目的是建立教會。保羅強調恩賜的來源和目的。

經文應用：

「恩賜原有分別，聖靈卻是一位。職事也有分別，主卻是一位。」（12:4-5）

ἀνάστασις (anastasis)

中文譯詞：復活

出現位置：第15章多處

字義解釋：原意是「站起來」，指從死裡復活。保羅強調基督的復活和信徒的復活是福音的核心，也是我們盼望的基礎。

經文應用：

「若基督沒有復活，你們的信便是徒然，你們仍在罪裡。」（15:17）

ἐλευθερία (eleutheria)

中文譯詞：自由

出現位置：10:29

字義解釋：基督徒在基督裡得著自由，但這自由要在愛心中運用，不可成為軟弱人的絆腳石。自由是為了服事神和愛人。

經文應用：

「凡事都可行，但不都有益處。凡事都可行，但不都造就人。」（10:23）

詳細屬靈意義分析

一、合一與分黨的屬靈教訓（1:10-4:21）

1. 分黨的本質

保羅指出，哥林多教會的分黨現象反映了屬靈的幼稚。當信徒高舉人的領袖過於基督，或將個人的屬靈經歷偶像化時，就會產生分黨。分黨不僅破壞合一，更顯明我們對福音本質的誤解。

「我的意思就是你們各人說：『我是屬保羅的』；『我是屬亞波羅的』；『我是屬磯法的』；『我是屬基督的』。基督是分開的嗎？保羅為你們釘了十字架嗎？」（1:12-13）

保羅的質問顯明：分黨就是分裂基督的身體，因為教會只有一個根基——基督。任何人或組織都不能成為教會的中心。

2. 十字架的反轉邏輯

保羅用「神的愚拙」和「神的軟弱」來顛覆世界的價值觀。十字架事件是歷史的轉折點，它徹底反轉了人類對智慧、能力、榮耀的定義。在十字架的光照下，所有屬人的成就都顯得蒼白無力。

世界的標準：

  • 智慧 = 知識和辯才
  • 能力 = 權力和影響
  • 榮耀 = 地位和名聲

神的標準：

  • 智慧 = 十字架的救恩
  • 能力 = 復活的大能
  • 榮耀 = 在軟弱中顯出

3. 屬靈成長的歷程

保羅將信徒分為「屬靈的」和「屬肉體的」，這不是地位的區分，而是生命成熟度的描述。屬肉體的人仍像嬰孩，只能吃奶，不能吃乾糧。屬靈的人能領受神的奧秘，有屬靈的洞察力。

屬靈成長的標記：

  • 從屬肉體到屬靈：不再以人的標準判斷，而是以神的眼光看事物
  • 從分黨到合一：不再高舉領袖，而是以基督為中心
  • 從爭競到配搭：明白各人都是神的僕人，要彼此配搭

二、聖潔與道德的屬靈原則（5:1-6:20）

1. 教會是聖潔的團體

保羅用「麵酵」的比喻說明罪的蔓延性。教會必須維護聖潔，因為她是基督的身體，是神的殿。容忍明顯的罪不僅傷害當事人，更會影響整個教會群體。

「你們這自誇是不好的。豈不知一點麵酵能使全團發起來嗎？你們既是無酵的麵，應當把舊酵除淨，好使你們成為新團。」（5:6-7）

教會的聖潔不是律法主義，而是反映我們在基督裡的新生命。我們是「新團」，理當除去舊的罪性。

2. 身體是聖靈的殿

保羅對身體的教導具有深遠的屬靈意義。身體不只是物質的，更是屬靈的。當我們信主時，聖靈就住在我們裡面，我們的身體就成了聖靈的殿。

消極意義：

不可與娼妓聯合，不可在身體上犯罪，因為這褻瀆了聖靈的殿。

積極意義：

要在身體上榮耀神，因為我們是重價買來的，是屬於神的。

「豈不知你們的身子就是聖靈的殿嗎？這聖靈是從神而來，住在你們裡頭的；並且你們不是自己的人，因為你們是重價買來的。所以，要在你們的身子上榮耀神。」（6:19-20）

三、愛與自由的屬靈平衡（7:1-11:1）

1. 知識與愛心的張力

在討論吃祭偶像之物時，保羅指出知識和愛心的區別。知識叫人自高自大，唯有愛心能造就人。基督徒的自由不是無限制的，而是在愛心中被規範的。

平衡的原則：

知識的層面：

  • 偶像算不得什麼
  • 食物不能叫神看中我們
  • 我們有自由

愛心的層面：

  • 要顧念軟弱的弟兄
  • 不可成為絆腳石
  • 寧可放棄自由

2. 自由與責任

保羅多次強調「凡事都可行」，但立刻加上「但不都有益處」、「但不都造就人」、「但無論哪一件，我總不受它的轄制」。自由是為了服事，而不是為了放縱。

「所以，你們或吃或喝，無論做什麼，都要為榮耀神而行。不拘是猶太人，是希臘人，是神的教會，你們都不要使他跌倒；就好像我凡事都叫眾人喜歡，不求自己的益處，只求眾人的益處，叫他們得救。」（10:31-33）

最高原則是「榮耀神」和「造就人」。自由不是目的，而是手段；不是權利，而是責任。

四、愛是最妙的道（12:1-13:13）

1. 愛的優先性

保羅在討論屬靈恩賜的脈絡中，插入了一篇關於愛的偉大篇章。這不是偶然，而是顯明：無論有多少恩賜，若沒有愛，都是虛空。愛是最大的恩賜，也是所有恩賜運用的動機和標準。

愛的十六個特質：

  • ✓ 恆久忍耐
  • ✓ 又有恩慈
  • ✓ 不嫉妒
  • ✓ 不自誇
  • ✓ 不張狂
  • ✓ 不做害羞的事
  • ✓ 不求自己的益處
  • ✓ 不輕易發怒
  • ✓ 不計算人的惡
  • ✓ 不喜歡不義
  • ✓ 只喜歡真理
  • ✓ 凡事包容
  • ✓ 凡事相信
  • ✓ 凡事盼望
  • ✓ 凡事忍耐
  • ✓ 永不止息

2. 愛與恩賜的關係

保羅指出，即使有說方言、先知講道、信心移山等大恩賜，若沒有愛，就「算不得什麼」（13:1-3）。愛不是一種恩賜，而是所有恩賜的基礎和目的。恩賜會過去，但愛永遠長存。

「如今常存的有信，有望，有愛這三樣，其中最大的是愛。」（13:13）

在永恆中，信心會成為眼見，盼望會成為實現，唯有愛是永不止息的。因此，愛是最大的。

五、復活的屬靈意義（15:1-58）

1. 復活是福音的核心

保羅將復活放在整卷書信的結尾，並用最長的篇幅論述。這不是偶然，因為復活是福音的核心，也是所有真理的根基。如果沒有復活，整個基督教信仰都是虛空。

復活的重要性：

  • 復活證明基督是神的兒子：「按聖善的靈說，因從死裡復活，以大能顯明是神的兒子。」（羅1:4）
  • 復活證明福音是真的：若基督沒有復活，我們所傳的便是枉然（15:14）
  • 復活是我們得救的保證：基督已經復活，我們也必復活
  • 復活是我們盼望的根基：「我們若靠基督只在今生有指望，就算比眾人更可憐。」（15:19）

2. 復活的次序

保羅用「初熟的果子」來形容基督的復活，說明復活的次序和關係。基督是第一個復活的，我們是他的跟隨者，也要復活。
