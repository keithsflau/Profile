# 聖經 2D 路線地圖 — Gemini 生成 Prompt 索引

> 參考風格：約書亞記得地進程圖、以色列分配產業地圖（教育性查經海報風格）
> 每個 prompt 已內嵌通用風格前綴，可直接全選複製至 **Gemini** 圖片生成。

## 使用方式

1. 打開 `prompts/` 資料夾內對應的 `.txt` 檔
2. 全選複製內容，貼到 Gemini
3. 可附上參考圖固定風格（建議用 04、05 號附圖）
4. 生成後存為同名 `.png`，放入本資料夾

## 使用建議

| 技巧 | 說明 |
|------|------|
| 附上參考圖 | 上傳約書亞得地或分配產業參考圖作風格參考 |
| 分步生成 | 太複雜時先「只生成地圖主體」，再加時間軸和資訊框 |
| 修正地理 | 生成後用文字要求修正地名位置 |
