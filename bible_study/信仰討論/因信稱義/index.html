因信稱義

Justification by Faith

恩典、信心與救恩的神學探討

引言：救恩的核心真理

「因信稱義」是基督教信仰的核心教義，也是宗教改革時期最重要的神學主題之一。這個教義回答了一個根本問題：人如何在神面前被視為義人？

這個問題不僅關乎我們對救恩的理解，更關乎我們與神的關係、我們對自己的認識，以及我們在信仰生活中的實踐。本討論將從聖經基礎、歷史發展、神學內涵，以及實際應用等層面，深入探討「因信稱義」這個重要的真理。

保羅以亞伯拉罕為例，說明稱義是基於信心，而非行為。亞伯拉罕在受割禮之前就已經因信稱義（羅馬書 4:9-11），這表明稱義是神白白的恩典，透過信心領受，與人的行為無關。

二、神學定義與內涵

何謂「稱義」？

「稱義」（Justification）是一個法律用語，意思是「宣告為義」。在神學上，稱義是指：

  • 神基於基督的救贖工作，宣告罪人為義
  • 神將基督的義歸算（impute）給相信的人
  • 這是一個法律地位的改變，而非道德品格的改變
  • 稱義是即時的、完全的，不是漸進的過程
🎁

恩典

稱義是神白白的恩典，不是人配得的，也不是人賺取的。這是神主動的作為，完全出於祂的慈愛和憐憫。

✝️

基督的救贖

稱義的基礎是基督在十字架上的代贖工作。基督代替我們承受罪的刑罰，使我們可以得著祂的義。

🙏

信心

信心是領受稱義的途徑。這不是一種功勞，而是接受神所預備的救恩。信心本身不是稱義的原因，而是工具。

義的歸算（Imputation）

基督的義歸算給我們，我們的罪歸算給基督。這是雙向的歸算，是稱義的核心機制。

三、不是因行為稱義

核心宣告：不是出於行為

提多書 3:5
「祂便救了我們；並不是因我們自己所行的義，乃是照祂的憐憫，藉著重生的洗和聖靈的更新。」

羅馬書 3:20
「所以凡有血氣的，沒有一個因行律法能在神面前稱義，因為律法本是叫人知罪。」

對比項目 因行為稱義（錯誤） 因信稱義（正確）
基礎 人的努力和功德 基督的救贖工作
途徑 遵守律法、行善 信心接受
性質 賺取、配得 恩典、白白的
結果 自誇、驕傲 感恩、謙卑
可能性 不可能（無人能完全守律法） 可能（靠基督的義）

1. 人的完全性問題

沒有人能夠完全遵守神的律法。雅各書 2:10 說：「因為凡遵守全律法的，只在一條上跌倒，他就是犯了眾條。」一個小過失就足以使我們在神面前成為不義。

2. 動機的問題

即使我們能夠行善，如果動機不純（為了賺取救恩、為了自誇），這些行為在神眼中仍然沒有價值。神看的是內心，而不僅僅是外在的行為。

3. 罪的嚴重性

罪是對無限聖潔之神的冒犯，需要無限的代價來償還。有限的人無論如何努力，都無法償還這個無限的債務。只有無限的神（基督）才能償還。

4. 神的標準

神的標準是絕對的完美。馬太福音 5:48 說：「所以，你們要完全，像你們的天父完全一樣。」這個標準只有基督達到，我們必須靠祂的義才能稱義。

四、信心與行為的關係

重要區分：稱義 vs 成聖

稱義（Justification）

  • ✓ 法律地位的改變
  • ✓ 即時的、完全的
  • ✓ 因信稱義，不靠行為
  • ✓ 一次性的宣告
  • ✓ 在神面前的地位

成聖（Sanctification）

  • ✓ 道德品格的改變
  • ✓ 漸進的、持續的
  • ✓ 需要行為配合
  • ✓ 一生的過程
  • ✓ 實際生活的改變

雅各書 2:14-26 的解釋

雅各書 2:17
「這樣，信心若沒有行為就是死的。」

雅各書強調「信心沒有行為是死的」，這似乎與保羅「因信稱義」的教導有衝突。但實際上，兩者是在不同層面上討論：

  • 保羅討論的是稱義的基礎：我們如何得救？答案是「因信稱義」，不是因行為。
  • 雅各討論的是信心的真實性：真正的信心必然會產生行為。沒有行為的信心是假的、死的信心。
  • 兩者並不矛盾：稱義是因信，但真信心必有行為。行為不是稱義的原因，而是稱義的證據。

信心與行為的正確關係

信心是稱義的途徑，行為是信心的證據。我們因信稱義，但真信心必然會產生好行為。

五、歷史發展與重要性

早期教會

使徒保羅在羅馬書和加拉太書中明確闡述「因信稱義」的教義，強調救恩是神的恩典，透過信心領受。

中世紀時期

教會逐漸偏離聖經的教導，強調行為和功德在救恩中的重要性，導致「因信稱義」的真理被模糊。

宗教改革（16世紀）

馬丁路德重新發現「因信稱義」的真理，這成為宗教改革的核心口號。他強調「唯獨信心」（Sola Fide），與「唯獨恩典」（Sola Gratia）和「唯獨基督」（Solus Christus）一起，成為改革宗神學的三大支柱。

現代意義

「因信稱義」至今仍然是基督教信仰的核心教義，提醒我們救恩完全是神的恩典，我們只能憑信心接受，不能靠行為賺取。

六、實際應用與生活意義

1. 得救的確據

因信稱義給我們得救的確據。我們不需要擔心自己是否做得夠好，因為救恩是基於基督的工作，不是我們的工作。我們可以確信，只要我們相信基督，我們就已經被稱義了。

2. 謙卑與感恩

認識到我們是因信稱義，而不是因行為稱義，會使我們謙卑。我們沒有什麼可誇的，一切都是神的恩典。這會產生深深的感恩，激勵我們用生命來回應神的愛。

3. 自由與釋放

因信稱義使我們從「必須靠行為賺取救恩」的捆綁中得釋放。我們不再需要活在恐懼和焦慮中，擔心自己是否做得夠好。我們可以自由地愛神、服事神，不是出於義務，而是出於感恩。

4. 好行為的動機

雖然我們不是因行為稱義，但我們仍然應該有好行為。好行為的動機不是為了賺取救恩，而是為了表達對神的感恩，順服神的命令，見證神的恩典，並在成聖的過程中與神同工。

⚠️ 常見的誤解與提醒

誤解一：「既然因信稱義，就不需要有好行為了。」

✓ 正確理解：真信心必然會產生好行為。雖然行為不是稱義的原因，但卻是信心的證據和成聖的表現。

誤解二：「信心是一種功勞，所以我們還是靠自己的努力。」

✓ 正確理解：信心本身不是功勞，而是接受神恩典的工具。連信心本身也是神所賜的（以弗所書 2:8）。

誤解三：「稱義後就可以隨意犯罪了。」

✓ 正確理解：這是保羅在羅馬書 6:1 所駁斥的錯誤觀念。稱義的人會追求成聖，因為他們已經與基督聯合，有新生命的樣式。

結論

「因信稱義」是基督教信仰的核心真理，它告訴我們：

  • 救恩完全是神的恩典，不是我們配得或賺取的
  • 稱義的基礎是基督的救贖工作，祂代替我們承受罪的刑罰
  • 我們透過信心接受這個救恩，信心是工具，不是功勞
  • 稱義是即時的、完全的，是法律地位的改變
  • 雖然我們不是因行為稱義，但真信心必然會產生好行為
  • 這真理給我們確據、謙卑、感恩和自由
