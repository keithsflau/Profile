# AI Generated Materials

A collection of educational materials and resources covering various subjects including:

- **Bible Study** - Biblical studies, character studies, and theological discussions
- **Mathematics** - Math resources and exercises for various grade levels
- **Science** - Biology, Chemistry, and Physics educational materials
- **Primary Chinese** - Chinese language learning resources and games

## Structure

- `bible_study/` - Biblical study materials and resources
- `Mathematics/` - Math educational content
- `Science/` - Science educational materials
- `PrimaryChinese/` - Chinese language learning portal and games
- `Hong_Kong_Tycoon_Dream/` - Game project

## Getting Started

This repository contains static HTML files and web-based educational resources that can be viewed directly in a web browser.
