# 詩詞捉迷藏 - Poetry Hide and Seek

A fun and colorful single-player game that gamifies the memorization of Tang Dynasty poetry.

## Features

- **Lobby System**: Enter your name and join 99 simulated bot players
- **Game Arena**: Answer poetry questions with a 5-second timer
- **Dynamic Activity**: Watch players find their matches each round (20% success rate)
- **Activity Feed**: Real-time feed showing player achievements
- **Victory Condition**: Complete 10 rounds to win the Scholar's Trophy
- **Happy Theme**: Bright and colorful design with blue, purple, and pink gradients

## Installation

```bash
npm install
```

## Development

```bash
npm run dev
```

## Build

```bash
npm run build
```

## Game Mechanics

1. Start with 100 players (you + 99 bots)
2. Each round presents a Tang Dynasty poetry question
3. Answer correctly to find the matching verse
4. 20% of remaining players find their match each round
5. Complete 10 rounds to win the Scholar's Trophy!

## Poetry Database

Includes 10 famous Tang Dynasty poem pairs:
- 舉頭望明月 / 低頭思故鄉
- 床前明月光 / 疑是地上霜
- 春眠不覺曉 / 處處聞啼鳥
- And 7 more...
