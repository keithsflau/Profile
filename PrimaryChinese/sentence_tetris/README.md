# 重組句子俄羅斯方塊 (Sentence Tetris)

中一中文學習遊戲 - 通過俄羅斯方塊風格遊戲學習中文複句

## 遊戲說明

這是一個教育性遊戲，讓學生通過類似俄羅斯方塊的玩法來練習構建複雜的中文句子。句子片段會從屏幕頂部落下，玩家需要將這些片段拖放到底部的「句子組合區」，並按正確的邏輯順序排列。

## 技術棧

- React 18
- Tailwind CSS
- Framer Motion (動畫和拖放)
- Vite (構建工具)

## 安裝和運行

```bash
# 安裝依賴
npm install

# 開發模式
npm run dev

# 構建生產版本
npm run build

# 預覽構建版本
npm run preview
```

## 遊戲特色

- **三種複句類型**：轉折複句、選擇複句、遞進複句
- **難度等級**：簡單和困難兩種速度
- **拖放功能**：可以拖動落下的方塊或堆積的方塊到組合區
- **視覺反饋**：正確時有清除動畫，錯誤時有震動效果
- **分數系統**：包含時間獎勵、連擊獎勵和關卡獎勵
- **音效預留**：已預留音效函數接口，方便後續添加音頻

## 遊戲玩法

1. 點擊「開始遊戲」開始
2. 句子片段會從頂部落下
3. 拖動片段到底部的「句子組合區」
4. 按正確順序排列所有片段
5. 點擊「發射/消除」提交答案
6. 如果正確，關卡清除並進入下一關
7. 如果錯誤，片段會震動並返回遊戲區域

## 關卡數據

遊戲包含三個預設關卡：
1. 雖然...但是... (轉折複句)
2. 與其...不如... (選擇複句)
3. 不但...而且... (遞進複句)

## 開發說明

音效函數已預留接口：
- `playClickSound()` - 點擊音效
- `playClearSound()` - 清除音效
- `playErrorSound()` - 錯誤音效

可以在 `App.jsx` 中替換這些函數來添加實際的音頻播放功能。
