# HKDSE Genetics Master

A powerful educational tool for HKDSE Biology students to master Mendelian Genetics.

## Features

### 1. The Punnett Calculator
- **Universal Support**: Handles Monohybrid (`Aa x Aa`), Dihybrid (`AaBb x aaBB`), and Sex-linked traits.
- **Smart Logic**: Automatically generates gametes and fills the Punnett Square.
- **Analysis**: Real-time calculation of Genotypic and Phenotypic ratios.

### 2. Pedigree Logic Trainer
- **Scenario Based**: Interactive challenges to deduce inheritance patterns.
- **Interactive Logic**: Test your hypothesis (e.g., "Is it Autosomal Dominant?") and get immediate, scientifically accurate feedback.
- **Visuals**: Clean, clear pedigree charts using standard symbols.

## Technology Stack
- **Framework**: React 19 + Vite
- **Styling**: Tailwind CSS
- **Charts**: Recharts
- **Icons**: Lucide React
- **Animations**: Framer Motion

## Installation

1. Clone the repository
2. Run `npm install`
3. Start dev server: `npm run dev`
