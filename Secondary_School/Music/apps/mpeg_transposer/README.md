# MPEG Transposer

A high-quality web application to transpose (change key) of audio files (MP3, WAV, etc.) and export them as MP3.

## Features

- **Pitch Shifting**: High-quality real-time pitch shifting (-12 to +12 semitones).
- **Format Support**: Import MP3, WAV, M4A, etc. Export as MP3.
- **Modern UI**: Sleek, glassmorphism design.
- **Client-Side**: All processing happens in your browser. No data leaves your computer.

## How to Run

1. Open this folder in VS Code.
2. Open a terminal.
3. Run `npm install` (if not already done).
4. Run `npm run dev`.
5. Open the Local URL (e.g., `http://localhost:5173`) in your browser.

## Technologies

- React + Vite
- Tone.js (Audio Processing)
- lamejs (MP3 Encoding)
- Tailwind CSS (Styling)
