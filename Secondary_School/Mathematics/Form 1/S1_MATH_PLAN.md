# Form 1 Mathematics Interactive Applications Plan

This document outlines the development plan for 30 interactive web applications for the Form 1 Mathematics curriculum (HKDSE aligned).
Current Status: Planning Phase

## Tech Stack

- **Core:** HTML5, Tailwind CSS
- **Logic:** React (CDN), Babel, JavaScript (ES6+)
- **Visuals:** Canvas API, Chart.js (for stats), FontAwesome Icons
- **Theme:** "Bright Math" - Clean, white/slate backgrounds with vibrant accent colors (Emerald, Blue, Indigo) to represent logic and precision.

## Application List

### Unit 1: Number Sense & Algebra

1. **01_Directed_Numbers**: `Directed Numbers Line` - Interactive number line for adding/subtracting positives and negatives.
2. **02_BODMAS_Calc**: `Order of Operations` - Step-by-step visual calculator showing BODMAS/PEDMAS rules.
3. **03_HCF_LCM**: `HCF & LCM Visualizer` - Venn diagrams and factor trees for common factors/multiples.
4. **04_Prime_Factors**: `Prime Factor Tree` - Interactive tree decomposition of numbers.
5. **05_Basic_Algebra**: `Algebra Tiles` - Visual representation of variables and constants.
6. **06_Substitution**: `Substitution Tool` - Plug in values X and Y to see evaluations.
7. **07_Sequences**: `Sequence Generator` - Visual number patterns (arithmetic/geometric).
8. **08_Function_Machine**: `Function Machine` - Input/Output machine for simple linear functions.
9. **09_Linear_Eq**: `Balance Scale Logic` - Solving simple equations ($x + 3 = 7$) using a balance scale.
10. **10_Metric_Converter**: `Metric Master` - Visual slider for converting lengths, areas, and volumes.

### Unit 2: Geometry & Measures

11. **11_Angles_Basic**: `Angle Explorer` - Dynamic protractor to measure acute, obtuse, reflex angles.
12. **12_Parallel_Lines**: `Parallel & Transversal` - Interactive lines showing corresponding, alternate, interior angles.
13. **13_Triangle_Angles**: `Triangle Sum` - Drag vertices to see angles always sum to 180.
14. **14_Area_Builder**: `Area & Perimeter` - Grid based builder for rectangles, triangles, parallelograms.
15. **15_Volume_Prism**: `Volume Visualizer` - Fill 3D prisms with unit cubes.
16. **16_Coordinates**: `Coordinate Game` - Battleship-style game for plotting (x, y) points.
17. **17_Transform_Slide**: `Translation Lab` - Slide shapes on a grid by vector.
18. **18_Transform_Reflect**: `Reflection Mirror` - Draw reflection across x-axis, y-axis, or arbitrary lines.
19. **19_Transform_Rotate**: `Rotation Spinner` - Rotate shapes about a point.
20. **20_3D_Nets**: `Net Unfolder` - Animate unfolding 3D shapes into 2D nets.

### Unit 3: Data & Others

21. **21_Percentage_Bar**: `Percentage Estimator` - Drag sliders to visualize percentages of amounts.
22. **22_Profit_Loss**: `Shopkeeper Sim` - Calculate discount, profit, and loss percentages.
23. **23_Ratio_Share**: `Ratio Divider` - Visual bar splitting amounts into ratios (e.g., 2:3).
24. **24_Pictogram**: `Pictogram Maker` - Drag icons to create pictograms.
25. **25_Bar_Chart**: `Bar Chart Builder` - Interactive tool to construct bar charts from data.
26. **26_Pie_Intro**: `Fraction Pie` - Relating fractions to pie chart sectors.
27. **27_Stat_Intro**: `Mean Mode Median` - Drag heights to visualize average vs median.
28. **28_Time_Zones**: `World Clock` - Calculate time differences.
29. **29_Estimation**: `Estimation Jar` - Guessing game to practice estimation strategies.
30. **30_Binary_Card**: `Binary Basics` - Introduction to number systems (enrichment).

## Development Guidelines

- All apps must be contained in a single `index.html`.
- Use `React` and `Tailwind CSS`.
- Ensure mobile responsiveness.
- **Strictly English** interface for S1 Science/Math (English Medium).
