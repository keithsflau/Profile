# 故事骰子生成器 - Story Dice Generator

一個創意寫作工具，幫助學生通過隨機生成的故事元素來激發創作靈感。

## 功能特色

- 🎲 **四個故事元素骰子**：人物、地點、物品、衝突
- 🎨 **精美的卡片動畫**：3D翻轉效果，視覺效果豐富
- 🏙️ **香港本地元素**：包含港鐵、茶餐廳、維多利亞港等本地特色
- ✍️ **故事寫作區域**：提供文字編輯器供學生創作
- 📋 **一鍵複製**：輕鬆將故事複製到剪貼簿

## 技術棧

- React 18
- Vite
- Tailwind CSS
- 純前端應用，無需後端

## 安裝與運行

```bash
# 安裝依賴
npm install

# 開發模式
npm run dev

# 構建生產版本
npm run build

# 預覽生產版本
npm run preview
```

## 使用說明

1. 點擊「擲骰子」按鈕
2. 等待卡片翻轉動畫完成
3. 查看四個類別隨機生成的故事元素
4. 在下方文字區域開始創作你的故事
5. 完成後點擊「複製到剪貼簿」保存作品

## 故事元素類別

### 人物 (Character)
包含12個創意角色，如「一個疲憊的戰士」、「一個在茶餐廳工作的年輕人」等。

### 地點 (Setting)
包含12個香港特色地點，如「廢棄的港鐵站」、「深夜的茶餐廳」、「維多利亞港的渡輪上」等。

### 物品 (Item)
包含12個神秘物品，如「一個壞掉的指南針」、「一杯無限的珍珠奶茶」等。

### 衝突 (Conflict)
包含12個故事衝突，如「不能說出真相」、「找不到回家的路」等。

## 開發說明

項目使用 Tailwind CSS 進行樣式設計，所有動畫效果均通過 CSS 實現，無需額外的動畫庫。

卡片翻轉效果使用 CSS 3D Transform 實現，提供流暢的視覺體驗。
