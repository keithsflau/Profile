# Shopping Quest - Vocabulary Game

Designed for a P3 Student to practice shopping vocabulary!

## How to Play
1. Open this folder in your customized terminal.
2. Run `npm install` (if not already done).
3. Run `npm run dev` to start the game string.
4. Click the link (usually http://localhost:5173) to play in the browser.

## Features
- **Adventure Mode**: Go through the Entrance, Fruit Section, Bakery, and more!
- **Visuals**: Fun icons for every word.
- **Spelling Practice**: Type the words to score points.
- **Boss Level**: "Sensible Shopper" challenge.
- **Confetti Rewards**: Celebrate every victory!

## Vocabulary Included
- **Fruit**: Bunch, Bananas, Fruit, Favourite
- **Bakery**: Bread, Cupcakes, Tarts, Delicious
- **General**: Entrance, Basket, List, Meat, Vegetables, Fish, Ice Cream
- **Bonus**: Sensible, Shopper
