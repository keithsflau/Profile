# Music Therapy - Iso-Mood Player

This is a prototype web application for music therapy using the **Iso-principle**. It generates music based on your current mood and gradually shifts it to a target mood.

## Features
- **Real-time Audio Synthesis**: Uses `Tone.js` to generate music on the fly.
- **Mood Entrainment**: Algorithms adapt tempo, pitch, and timbre to match and guide emotions.
- **Visual Feedback**: Reactive animations based on the audio.
